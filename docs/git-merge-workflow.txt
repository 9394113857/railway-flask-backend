üî• GIT MERGE WORKFLOW (FEATURE BRANCH ‚Üí MAIN)
=========================================================
This is the standard flow followed in real projects.
Save this file. Follow these steps EVERY TIME you work.
=========================================================
---------------------------------------------------------
STEP 0 ‚Äî Make sure you are on your feature branch
---------------------------------------------------------

git branch # Shows current branch
git switch feature/sqlalchemy-migrations # OR git checkout

---------------------------------------------------------
STEP 1 ‚Äî Add & Commit all changes in feature branch
---------------------------------------------------------

git add .
git commit -m "Completed migrations, API fixes, and testing"

---------------------------------------------------------
STEP 2 ‚Äî Push feature branch to remote
---------------------------------------------------------

git push origin feature/sqlalchemy-migrations

---------------------------------------------------------
STEP 3 ‚Äî Switch to MAIN branch
---------------------------------------------------------

git switch main # OR git checkout main

---------------------------------------------------------
STEP 4 ‚Äî Sync your local MAIN with remote MAIN
(always do this before merging)
---------------------------------------------------------

git pull origin main

---------------------------------------------------------
STEP 5 ‚Äî Merge feature branch into main
---------------------------------------------------------

git merge feature/sqlalchemy-migrations
